## Getting started

1. Install dependencies and start.

   ```bash
   yarn
   yarn dev
   ```

2. Start strapi server. ([read more](./api/README.md))

   ```bash
   cd ./api
   yarn start
   ```

3. Build for production.

   ```bash
   yarn build
   ```

4. Test your production build.

   ```bash
   yarn serve
   ```
